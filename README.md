## ‚ú® Overview

**Imagine** is a modern, material-you designed Android application for all your image editing needs. Whether you need to resize a photo, remove a background, edit EXIF data, or generate a color palette, Imagine handles it all‚Äî**entirely on-device**. No cloud uploads, no privacy concerns.

Designed for both **casual users** and **power users**, Imagine combines a beautiful UI with professional-grade tools.

## üöÄ Key Features

### üé® Editing & Manipulation
*   **Filters**: Apply over **300+** stunning filters and effects.
*   **Cropping**: Advanced cropping with aspect ratios, free rotation, and smart shape masks (Circle, Star, Heart, etc.).
*   **Draw & Mark**: Annotate images with a powerful drawing engine (Pen, Neon, Highlighter, Privacy Blur).
*   **Background Removal**: AI-powered automatic background erasure + manual precision tools.
*   **Transform**: Resize, scale, flip, and rotate with various high-quality algorithms (Lanczos, Bicubic, etc.).

### üõ†Ô∏è Utilities
*   **Format Conversion**: Convert between JPG, PNG, WEBP, HEIC, AVIF, JXL, PDF, and more.
*   **Compression**: Smart image shrinking to specific file sizes (e.g., "Compress to 500kb").
*   **Privacy**: Edit and strip **EXIF metadata** (GPS, Camera model) to protect your privacy.
*   **OCR**: Extract text from images using advanced on-device recognition (120+ languages).
*   **Colors**: Extract dynamic color palettes from any image or generate Material You themes.

### üõ°Ô∏è Privacy & Security
*   **Offline First**: All processing happens on your device.
*   **Encryption**: Secure your sensitive images with military-grade encryption (AES-256).
*   **Privacy Blur**: Quickly redact sensitive info from screenshots.
